# Agro Forte, Futuro Sustentável 🌱

Este é um projeto de site interativo desenvolvido a partir de um protótipo do Figma. O objetivo da página é conscientizar os usuários sobre a importância do *Agro Forte e Sustentável*, mostrando como o equilíbrio entre a produção no campo e a preservação do meio ambiente é fundamental para o futuro.

A aplicação conta com uma interface dinâmica onde o usuário pode navegar por informações essenciais, receitas do campo com aproveitamento integral de alimentos e um quiz interativo de múltipla escolha.

---

## 🚀 Funcionalidades

* *Design Fiel ao Figma:* Layout estruturado exatamente como o protótipo, utilizando a paleta de cores original (tons de verde e marrom).
* *Menu Lateral Interativo:* Botões totalmente funcionais que atualizam a área central da tela sem a necessidade de recarregar a página.
* *Ilustração SVG Sob Medida:* Uma imagem moderna inserida direto no código que representa a união da tecnologia agrícola com a natureza.
* *Livro de Receitas do Campo:* Aba com receitas práticas utilizando ingredientes do campo e focado no desperdício zero (uso de cascas e talos).
* *Quiz Dinâmico:* Um jogo de perguntas e respostas com 5 questões sobre sustentabilidade. O sistema valida as respostas na hora, mudando a cor dos botões para Verde (Correto) ou Vermelho (Incorreto).
* *Responsividade:* O site se adapta automaticamente para telas menores, como celulares e tablets.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando as tecnologias web fundamentais, sem a necessidade de instalar frameworks externos:

* *HTML5:* Estruturação semântica da página.
* *CSS3:* Estilização, definição da paleta de cores, layout em Flexbox e animações de transição suave.
* *JavaScript (ES6):* Lógica do quiz, manipulação dinâmica do conteúdo do quadrado central e controle de estado dos botões.
* *SVG (Scalar Vector Graphics):* Renderização da arte central diretamente pelo navegador, garantindo leveza e alta resolução.
